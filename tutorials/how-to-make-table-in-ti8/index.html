Step 1: Enter Your Function in Y= Menu

Press Y=, select Y1, type your function (e.g., x^2 + 3x), and press ENTER. You can enter multiple functions in Y2, Y3, etc., all of which will appear in the table.

Step 2: Open the Table

Press 2ND + GRAPH to open the TABLE. The x-values appear on the left, y-values on the right. This instant display helps visualize function behavior quickly.

Step 3: Adjust Table Settings

Press 2ND + WINDOW (TBLSET). Set TblStart for where the table begins and ŒîTbl for step size (1 for standard, 0.1 for fine, 5 for large jumps). Press TABLE again to see changes.

Step 4: AUTO vs ASK Mode

In TBLSET, choose AUTO (x-values filled automatically) or ASK (enter specific x-values manually). Use ASK for points like x = 2.5, AUTO for general function exploration.

Step 5: Edit or Clear a Function

Go to Y=, press CLEAR on old function, type a new one, and return to TABLE. Easy for switching problems or exploring multiple equations.
